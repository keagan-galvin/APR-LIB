Static Members
▸ toast(options)
toast(options: Object): Promise<string>
Parameters
options (Object)
Name Description
options.message string Message to be displayed.
options.duration number (default 2000) milliseconds of time to display toast. Set to null/undefined to display indefinitely.
options.actions Array<{label: string, classes: string, value: string}> (default [{label:'ok'}]) Toast Actions.
options.classes Array<string> (default []) Additional classes to be applied to the Toast Element.
Returns
Promise<string>: The clicked actions value will be sent in the returned Promise's resolution. If no value is provided, label value will be sent. If the toast is closed by the duration timer, "timeout" will be sent.
